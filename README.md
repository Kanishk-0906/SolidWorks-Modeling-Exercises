# SolidWorks-Modeling-Exercises

This repository contains a collection of mechanical components, assemblies, and modeling exercises created while learning and improving CAD design skills in SolidWorks.

The models cover a range of topics including:

* Part modeling
* Assembly design
* Parametric modeling
* Design intent and feature-based modeling
* Mechanical components and practice exercises

The purpose of this repository is to document my CAD learning journey and showcase progressive improvement in mechanical design and modeling techniques.

## Software Used

* SolidWorks
